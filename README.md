<h1>Muryo Mangás</h1>
<h2><strong>Descrição:</strong></h2>
    <p>O Muryo Mangás é uma aplicação web desenvolvida como projeto da disciplina de Desenvolvimento Web I, utilizando o framework Astro. O sistema consiste em uma loja virtual de mangás criada para oferecer uma experiência simples, organizada e intuitiva para os usuários, permitindo navegar pelo catálogo, visualizar detalhes dos produtos, gerenciar um carrinho de compras e acessar informações sobre a empresa.<br>
    O projeto teve como principal objetivo aplicar, na prática, os conhecimentos adquiridos durante a disciplina sobre desenvolvimento web moderno, explorando os recursos oferecidos pelo Astro para criação de páginas estáticas com componentes reutilizáveis, organização do código e boa experiência de navegação.<br>
    Além do aprendizado técnico, o tema foi escolhido por representar um universo bastante presente entre o público jovem e em constante crescimento. Dessa forma, foi possível unir o estudo de tecnologias web ao desenvolvimento de um sistema relacionado a um assunto de interesse do grupo, tornando o projeto mais motivador e próximo da realidade.</p>
<h2><strong>Objetivos do Projeto</strong></h2>
    <p>Este projeto teve como finalidade aplicar os conceitos estudados na disciplina de Desenvolvimento Web I, aprender o funcionamento do framework Astro, desenvolver uma aplicação web organizada utilizando componentes, integrar HTML, CSS e JavaScript em um projeto real, compreender a organização de páginas, layouts e reutilização de código e demonstrar que é possível unir tecnologia e interesses pessoais em um projeto prático.</p>

<h2><strong>Tecnologias Utilizadas:</strong></h2>
    <p>O desenvolvimento da aplicação foi realizado utilizando as seguintes tecnologias e ferramentas:</p>
    <ul>
        <li>Astro;</li>
        <li>HTML;</li>
        <li>CSS;</li>
        <li>JavaScript;</li>
        <li>GitHub;</li>
        <li>Vercel.</li>
    </ul>

<h2><strong>Estrutura do Sistema</strong></h2>
    <p>A interface foi desenvolvida seguindo um padrão visual único em todas as páginas, utilizando uma paleta de cores inspirada na identidade visual da Muryo Mangás. O sistema possui componentes reutilizáveis, como menu de navegação, barra de pesquisa e rodapé, garantindo maior organização e facilidade de manutenção.<br>
    As páginas implementadas são:</p>
    <ul>
        <li>Login de usuários e administradores;</li>
        <li>Catálogo de mangás;</li>
        <li>Página individual de cada produto;</li>
        <li>Carrinho de compras;</li>
        <li>Perfil do cliente;</li>
        <li>Página "Sobre Nós";</li>
        <li>Painel administrativo.</li>
    </ul>

<h2><strong>Desafios Encontrados</strong></h2>
    <p>Durante o desenvolvimento alguns recursos exigiram maior atenção por envolverem manipulação dinâmica de dados e interação entre diferentes páginas da aplicação.<br>
    Os principais desafios foram:</p>
    <ul>
        <li>Implementação das sugestões na barra de pesquisa;</li>
        <li>Desenvolvimento do carrinho de compras utilizando Local Storage;</li>
        <li>Atualização automática do resumo da compra;</li>
        <li>Organização dinâmica dos produtos adicionados ao carrinho;</li>
        <li>Diferenciação entre telas de administrador e cliente;</li>
        <li>Implementação da navegação para páginas individuais dos produtos;</li>
        <li>Gerenciamento dos dados armazenados localmente.</li>
    </ul>

<h2><strong>Melhorias Futuras</strong></h2>
    <p>Como continuidade do projeto, foram identificadas diversas funcionalidades que podem tornar o sistema ainda mais completo:</p>
    <ul>
        <li>Botão "Visualizar como Cliente" no painel administrativo;</li>
        <li>Funcionamento completo do histórico de compras;</li>
        <li>Sistema para criação de comentários pelos usuários;</li>
        <li>Recomendações de mangás baseadas no produto visualizado pelo usuário;</li>
        <li>Atualização do preço do produto conforme a quantidade selecionada antes da adição ao carrinho;</li>
        <li>Remoção parcial da quantidade de um produto no carrinho;</li>
        <li>Divulgação do projeto de doação por meio de banners e anúncios na página inicial;</li>
        <li>Tornar a logo clicável para retornar ao catálogo;</li>
        <li>Integração com uma API de mangás para obtenção dinâmica das informações dos produtos;</li>
        <li>Implementação de um back-end para autenticação, armazenamento de usuários, produtos e pedidos.</li>
    </ul>

<h2><strong>Considerações Finais:</strong></h2>
    <p>O desenvolvimento do Muryo Mangás proporcionou uma experiência prática na construção de uma aplicação web utilizando Astro, permitindo consolidar conhecimentos sobre componentes, estilização, manipulação do DOM e armazenamento local de dados. Além de atender aos objetivos da disciplina, o projeto serviu como base para futuras evoluções, podendo ser expandido com funcionalidades mais avançadas, integração com banco de dados e consumo de APIs, aproximando-o de uma aplicação utilizada em um cenário real.</p>
